# Lab 4 - AR using MediaPipe

## Aim
To develop an Augmented Reality (AR) application using webcam-based face tracking.

## Technologies Used
- MediaPipe FaceMesh
- JavaScript
- HTML5 Canvas

## Description
This application captures real-time video from the webcam and detects facial landmarks using MediaPipe. These landmarks are displayed as red points on the face, creating an AR overlay effect.

## How to Run
1. Open the project in VS Code
2. Right-click on index.html
3. Select "Open with Live Server"
4. Allow camera access
